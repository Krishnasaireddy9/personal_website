# Krishna Sai Reddy — Portfolio

Single-page React/Vite portfolio built with Tailwind CSS, Framer Motion, react-scroll, Lucide, EmailJS, and yet-another-react-lightbox.

## Run locally

```bash
npm install
npm run dev
```

## Personalize

- Replace `#` social/project URLs in the components and `src/data/projects.js`.
- Add `public/resume.pdf`.
- Replace `public/photos/photo-*.svg` with your own images and update `src/data/site.js` paths.
- Set the three EmailJS constants at the top of `src/components/Contact.jsx`.

Deploy the repository directly to Vercel; its Vite preset requires no additional configuration.
